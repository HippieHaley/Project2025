How to Use Note Reformatter

Edit the text in the top box how you want it, click format & copy to get the EMR ready note with multiple lines. :)

  1. Type or Paste: Enter your notes in the editor box below.
  2. Format Text: Select text and use the toolbar buttons to apply bold, italic, underline, colors, etc.
  3. Font Color: Select text and use the "Font" color picker to change text color.
  4. Highlight: Select text and use the "Highlight" color picker to add background color.
  5. Line Breaks: Press Enter to create new lines/paragraphs.
  6. Automatic Date Coloring: When you click "Format & Copy HTML", dates will automatically be color-coded:
    • Red = Past due
    • Orange = Due within 3 months
  7. Copy HTML: The formatted HTML code is automatically copied to your clipboard and displayed below for pasting into your EMR.
  8. Empty Template: If the editor is empty, it creates a template with ARU, AE, Pap, Mammo, and Depo labels.

Tip: All your formatting is preserved in the HTML output!
